Audio Processing

AVFoundation Framework

Apple's comprehensive audio/video framework providing:

  • AVAudioRecorder - High-quality audio recording functionality with metering support
  • AVAudioPlayer - Audio playback with equalizer processing capabilities
  • AVAudioEngine - Real-time audio processing and node graph management
  • AVAudioPlayerNode - Audio node for playback scheduling and control
  • AVAudioUnitEQ - 14-band dual-channel parametric equalizer implementation
  • AVAudioUnitTimePitch - Time-stretching and pitch-shifting effects

CoreAudio

Low-level audio processing framework for advanced audio manipulation and performance optimization.

Design Patterns

Singleton Pattern

  • APIManager - Single instance for authentication and data management
  • Audio - Global audio processing singleton

Benefits: Ensures single audio session, consistent auth state, simplified global access

Delegation Pattern

  • AVAudioRecorderDelegate - Recording lifecycle event handling
  • GIDSignInDelegate - Google Sign-In callbacks
  • GIDSignInUIDelegate - Google Sign-In UI presentation
  • UITableViewDelegate / UITableViewDataSource - Table view management

Notification Pattern

  • NotificationCenter - Cross-component communication
    • Authentication state changes
    • View controller navigation
    • Loosely-coupled component interaction

Extension Pattern

  • UIColor extension - Application color palette
  • UIFont extension - Typography system
  • UINavigationBar extension - Custom styling
  • Array extension - Utility functions
